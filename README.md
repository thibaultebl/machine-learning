# Linear regression closed form and deep neural network structure

## 📌 Motivation

I aimed to build at first a basic linear regression component made of the closed form, as a simple introduction to ML.
Then I decided to implement a neural network able to hold many layers, working on optimization and having a code object-oriented, in order to understand deeper what happen under the hood. 


## 🗂️ Project Structure
```
├── DNN/           # Deep Neural network structure 
├── linearregression/     # linear regression instance, operate the matrix operations and compute the beta coefficient  
├── utils/          # Data splitter, normalizer and csv reader. 
└── README.md
```
